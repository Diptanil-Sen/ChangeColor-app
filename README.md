# 🎨 ChangeColor App

A simple Android application built using **Kotlin** and **Android Studio** that changes the background color randomly whenever the user taps the Floating Action Button.

## 📱 Features

- Random background color generation
- Clean and simple user interface
- Material Design Floating Action Button
- Built using Kotlin
- Beginner-friendly Android project

## 🛠️ Technologies Used

- Kotlin
- Android Studio
- Android SDK
- ConstraintLayout
- Material Components

## 🚀 How It Works

1. Launch the application.
2. Tap the Floating Action Button in the center of the screen.
3. The background color changes randomly from a predefined list of colors.

## 🔧 Installation

1. Clone the repository:

```bash
git clone https://github.com/Diptanil-Sen/ChangeColor-app.git
```

2. Open the project in Android Studio.

3. Sync Gradle files.

4. Run the app on an emulator or physical Android device.

- Event Handling with `setOnClickListener`
- Random Color Generation
